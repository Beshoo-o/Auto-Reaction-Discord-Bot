## 👋🏻 Hi this is a Auto Reaction Discord Bot.

## ⚙️ Code Language :
- **JavaScript**

## 🚀 Getting Started :
- Install [NodeJs](https://nodejs.org/en)
- Extract the files
- Open the console
- Type ```bach npm install```
- Configure the Config.js
- Run the bot by clicking Start.cmd
